# MNIST 手写数字识别系统

基于 PyTorch 实现的手写数字识别系统，支持图像预处理、模型训练与预测功能，包含卷积神经网络(CNN)和全连接网络两种模型对比。

## 项目简介

本项目实现了一个完整的手写数字识别流程，主要功能包括：
- 自动训练适用于 MNIST 数据集的深度学习模型
- 对输入图像进行增强预处理，适配 MNIST 数据格式
- 基于训练好的模型进行数字识别并可视化结果
- 支持调试模式，可查看预处理各步骤效果

系统包含两种模型架构（CNN 和简单全连接网络），便于对比不同网络结构的识别性能。

## 功能说明

### 1. 模型架构

#### 卷积神经网络 (CNN)
- 两层卷积层：提取图像空间特征
- 最大池化层：减小特征图尺寸，增强鲁棒性
- Dropout层：防止过拟合
- 两层全连接层：最终分类输出

#### 简单全连接网络 (SimpleFC)
- 仅包含两层全连接层
- 用于对比卷积网络的性能差异

### 2. 图像预处理流程

1. 灰度转换：将彩色图像转为灰度图
2. 边缘裁剪：去除图像周围空白区域
3. 尺寸调整：缩放到20×20像素
4. 居中填充：扩展至28×28像素（MNIST标准尺寸）
5. 二值化：增强黑白对比度
6. 反色处理：确保数字为白色，背景为黑色
7. 张量转换与归一化：适配模型输入格式

### 3. 使用方法

#### 首次运行（自动训练模型）python main.py系统会自动检测是否存在预训练模型，若不存在则启动训练流程，默认训练CNN模型（15个epoch）

#### 识别自定义图像
1. 将待识别图像放入指定目录（默认：`test number`）
2. 在`main()`函数中修改`image_path`参数指向目标图像
3. 运行程序即可得到识别结果

#### 调试模式
开启调试模式可查看预处理各步骤效果：recognize_handwritten_digit(image_path, debug=True)

## 性能说明

- CNN模型在MNIST测试集上准确率可达99%以上
- 全连接模型准确率约为97%左右
- 预处理流程可有效提升对非标准手写数字的识别能力

## 使用示例

1. 准备一张手写数字图像（如`test number/0_image.jpg`）
2. 运行程序，系统会自动：
   - 检查并训练模型（首次运行）
   - 预处理输入图像
   - 输出识别结果及概率
